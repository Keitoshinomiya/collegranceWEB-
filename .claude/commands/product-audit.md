# 商品情報正確性チェック

## 対象
$ARGUMENTS（ブランド名、商品名、または「全件」。指定なければ画像あり商品を全件チェック）

## チェック手順

### 1. 画像の目視確認
`products.json` の画像がある商品について、実際の画像ファイルを Read ツールで表示し、以下を確認:
- ボトルの色（クリア/黒/色付き）が商品名・濃度と一致するか
- ラベル・ブランドロゴが正しいか
- 容量表記と一致するか（同じ香水でもサイズでボトルデザインが変わることがある）
- EDT/EDP でパッケージが異なるブランドがある（例: DIPTYQUE の EDP は黒ボトル）

### 2. 既知の要注意パターン
以下のブランドは容量・濃度でボトルが変わるため特に注意:
- **DIPTYQUE**: EDT = クリアボトル、EDP = 黒ボトル（75ml）
- **Jo Malone**: Cologne = クリア、Intense = 黒ボトル
- **TOM FORD**: 通常 = 金キャップ、Private Blend = 別デザイン
- **Maison Margiela**: Replica シリーズは全て統一デザインだが限定版あり
- **Le Labo**: サイズによってラベルデザインが異なる

### 3. Web検索で実物を確認
画像に疑わしい点がある場合:
- WebSearch で「{ブランド} {商品名} {容量} {濃度} bottle」を検索
- 公式サイトまたは信頼性の高い香水サイト（Fragrantica等）の画像と比較
- 差異があればレポートに記載

### 4. データ整合性チェック
- `concentration` が正しいか（EDT/EDP/Cologne/Parfum）
- `size` が一般的な展開サイズか（存在しないサイズでないか）
- `notes` が正しいか（Fragrantica等で確認）
- 同じ商品の重複がないか（サイズ違いは除く）
- `samplePrice` が相場と乖離していないか
