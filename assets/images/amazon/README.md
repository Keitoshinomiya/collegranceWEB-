# Amazon画像 統合フォルダ

## 構造

```
amazon/
├── _common/        共通PT画像（PT01/03/04/05/07/08）
├── scents/         香り別素材（main, PT02, PT06）
├── cases/          ケース色素材
└── variations/     SKU別最終画像（Amazon掲載URL用）
```

## ボリューム

- 22香水（001マン除外）
- 8香水: ONLY + 10色 = 11フォルダ × 8 = 88
- 14香水: ONLY のみ = 14
- 合計102フォルダ

- メイン画像: https://collegrance.com/assets/images/amazon/variations/014/BLK/main.jpg
- スウォッチ: https://collegrance.com/assets/images/amazon/variations/014/BLK/swatch.jpg
