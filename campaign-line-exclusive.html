Netlify 環境変数の設定

Netlify → Site config → Environment variables に追加：
ADMIN_SECRET = このパネルのパスワード
LINE_CHANNEL_ACCESS_TOKEN = LINE Developers で発行した長期トークン
LINE_CHANNEL_SECRET = LINE Developers のチャネルシークレット
LINE_TEST_USER_ID = テスト送信先のLINE User ID（自分）
BROADCAST_ENABLED = true（スケジュール自動送信を有効化）

LINE Broadcast — Messaging API

※ テスト送信は LINE_TEST_USER_ID に設定されたユーザーにのみ送信されます。
※ スケジュール自動送信は毎日 JST 12:00 に実行（BROADCAST_ENABLED=true のとき）。
※ 本番: netlify.toml の schedule を "0 15 28 3 *" に変更し BROADCAST_ENABLED=true を設定。
