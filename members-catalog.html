Setup Guide
Stripe 本番設定手順
1. Stripe Dashboard でアカウント作成
2. 各商品の Price ID（price_xxx）を取得
3. このファイルの STRIPE_PK に公開キーを貼り付け
4. 各商品の stripePriceId を実際の Price ID に更新
5. Netlify の環境変数に秘密キー（STRIPE_SK）を設定
6. パスワードを変更（MEMBER_PW）
